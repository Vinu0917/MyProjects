Introduction

Java is a powerful, versatile, and platform-independent programming language designed to build robust applications.

Created by James Gosling at Sun Microsystems in 1995, Java follows the principle of "Write Once, Run Anywhere" (WORA).

Key features of Java include:

  • Object-Oriented Programming
  • Platform Independence
  • Automatic Memory Management
  • Rich Standard Library
  • Security Features
// Your first Java program public class HelloWorld { public static void main(String[] args) { System.out.println("Hello, Java!"); } }
Getting Started

To begin Java development, you need to:

1. Download and install the Java Development Kit (JDK)

2. Set up environment variables (JAVA_HOME)

3. Choose an Integrated Development Environment (IDE)

// Check Java version java -version // Compile a Java program javac HelloWorld.java // Run a Java program java HelloWorld
Basic Syntax

Java syntax is clear and consistent, following C/C++ style conventions.

Understanding basic syntax is crucial for writing Java programs:

  • Case sensitivity
  • Class names start with uppercase
  • Method names start with lowercase
  • Source file name must match class name
public class Calculator { private int result; public int add(int a, int b) { result = a + b; return result; } }
Object Oriented Programming

Java is fundamentally object-oriented, built around the concept of objects containing data and code.

Core OOP concepts in Java include:

  • Encapsulation
  • Inheritance
  • Polymorphism
  • Abstraction
// Example of inheritance public class Animal { protected String name; public void makeSound() { System.out.println("Some sound"); } } public class Dog extends Animal { @Override public void makeSound() { System.out.println("Woof!"); } }
Exception Handling

Java provides robust exception handling mechanisms to deal with runtime errors.

Exception handling involves try, catch, finally, throw, and throws keywords.

public class ExceptionExample { public static void main(String[] args) { try { int result = divide(10, 0); } catch (ArithmeticException e) { System.out.println("Cannot divide by zero!"); } finally { System.out.println("Cleanup code"); } } public static int divide(int a, int b) { return a / b; } }
Collections Framework

Java Collections Framework provides a unified architecture for storing and manipulating groups of objects.

Main interfaces in the collections framework:

  • List - Ordered collection
  • Set - Unique elements
  • Queue - FIFO structure
  • Map - Key-value pairs
// Collections examples List list = new ArrayList<>(); Set set = new HashSet<>(); Map map = new HashMap<>(); list.add("Java"); set.add(42); map.put("key", 100);
Multithreading

Java provides built-in support for multithreaded programming.

Threads allow concurrent execution of two or more parts of a program.

class MyThread extends Thread { public void run() { System.out.println("Thread is running"); } } // Using Runnable interface class MyRunnable implements Runnable { public void run() { System.out.println("Runnable is running"); } }
File IO

Java provides comprehensive support for file input and output operations.

Common I/O classes include FileInputStream, FileOutputStream, FileReader, and FileWriter.

// Reading from a file try (BufferedReader reader = new BufferedReader(new FileReader("file.txt"))) { String line; while ((line = reader.readLine()) != null) { System.out.println(line); } } catch (IOException e) { e.printStackTrace(); }
Best Practices

Following best practices ensures maintainable and efficient Java code:

  • Follow naming conventions
  • Write clear documentation
  • Handle exceptions properly
  • Use appropriate access modifiers
  • Implement proper resource management
// Example of good practices public class UserService { private final UserRepository userRepository; public UserService(UserRepository userRepository) { this.userRepository = userRepository; } public User findUserById(Long id) throws UserNotFoundException { return userRepository.findById(id) .orElseThrow(() -> new UserNotFoundException(id)); } }
